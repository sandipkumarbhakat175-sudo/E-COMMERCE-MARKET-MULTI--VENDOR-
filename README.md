# PROJECT NAME- E-COMMMERCE-MARKET-MULTI-VENDOR

# PROJECT DETAIL-
A modern, responsive multi‑vendor e‑commerce marketplace built with HTML, CSS, and JavaScript, inspired by platforms like Amazon and Flipkart. This project demonstrates how multiple vendors can showcase products in a unified marketplace with interactive features.

🚀 Features
1. Multi‑Vendor Support – Vendors can list products independently.

2. Responsive Design – Optimized for desktop and mobile devices.

3. Product Cards – Display product image, price, discount, quantity, and rating.

4. Interactive UI – Hover effects, sticky navigation, and dynamic product grid.

5.Hero Banner – Eye‑catching promotional section with call‑to‑action.

6.Shopping Cart Simulation – Add to cart buttons with styled interactions.

7.Professional Layout – Clean design inspired by Amazon/Flipkart aesthetics.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript

Design: Responsive grid, flexbox, modern UI components

Styling: Custom CSS with hover animations and transitions

🎯 Purpose
1. This project is ideal for:

2. Learning frontend development through a real‑world use case.

3. Understanding multi‑vendor marketplace structure.

4.Building a foundation for a full‑stack e‑commerce application.
